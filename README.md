## متغيرات البيئة المطلوبة

```
DISCORD_TOKEN=   # توكن البوت (مطلوب)
BOT_APP_ID=      # معرف التطبيق (اختياري)
YOUTUBE_API_KEY= # لنظام صناع المحتوى (اختياري)
```

## التشغيل

```bash
cd bots/obt-system
python main.py
```

## الإعداد الأولي في السيرفر

1. أضف البوت لسيرفرك
2. استخدم `/إعدادات لوق` لإعداد قنوات اللوقات
3. استخدم `/إعدادات لوق_رئيسي` لتعيين القناة الرئيسية للأحداث
4. استخدم `/تذاكر إعداد` ثم `/تذاكر لوحة` لنظام التذاكر
5. استخدم `/حماية تفعيل` لتفعيل أنظمة الحماية
6. استخدم `/إعداد_ترحيب` و `/إعداد_وداع` للرسائل التلقائية
.
